# MongoPOS

MongoPOS is a modern, efficient Point of Sale system for Windows, macOS, and Linux. Built with performance and simplicity in mind — perfect for small and medium-sized businesses.

## 📋 **Documented Features:**

### **Core Modules:**
- **Product Management** – Catalog, categories, barcodes, images
- **Customer Management** – Profiles, history, database
- **Sales System** – Cart, payments, receipt printing
- **Returns** – Full return handling system
- **Reports & Analytics** – Sales, inventory, Excel export
- **Printing System** – Tickets, labels, barcode printing
- **User Management** – Authentication, roles, permissions
- **Settings** – Company and system configuration

### **Key Highlights:**
- 💻 Modern, responsive interface
- 🔔 Real-time notification center
- 📦 Advanced inventory and barcode handling
- ⚡ High-performance native desktop app powered by **Wails + Golang**
